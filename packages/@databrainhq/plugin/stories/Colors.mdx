# Usage

All color values are attached to root element as CSS variables and made availabe through tailwind classes. <br />
**Availabe values:** <br />
`primary` `primary-dark` `secondary` `secondary-dark` `white` `gray` `gray-dark` `alert-light` `alert` `alert-dark` `success` `success-dark` `info` `info-light`

**Example:**

```css
div {
  @apply bg-primary-dark text-primary;
}

.alert {
  @apply bg-alert text-alert-dark;
}
```

```html
<div className="bg-gray text-primary">Hello world</div>
```

**Note** For text color please use `text-primary` or `text-secondary-dark` and for other color values use `bg-primary-dark` or `bg-secondary`.
